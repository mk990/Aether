# Aether

### اینترنت آزاد برای همه :))

**[🇬🇧 English README](README.md)** · **[📖 مستندات کامل](Docs/GUIDE.fa.md)**

Aether یک کلاینت دور زدن سانسور اینترنت است که برای شبکه‌های به‌شدت محدود و فیلترشده طراحی شده است. این برنامه به‌صورت خودکار مسیرهای قابل‌دسترس را پیدا می‌کند، یک تونل رمزنگاری‌شده برقرار می‌کند و یک پراکسی محلی SOCKS5 در اختیار برنامه‌های شما قرار می‌دهد.

برخلاف VPNهای سنتی، Aether برای محیط‌هایی ساخته شده که در آن‌ها بازرسی عمیق بسته‌ها (DPI)، شناسایی الگوی پروتکل، محدودسازی UDP و مسدودسازی مقصدها به‌طور گسترده انجام می‌شود.

## ویژگی‌ها

- کشف خودکار بهترین مسیر
- پشتیبانی از MASQUE (HTTP/3 و HTTP/2)
- پشتیبانی از WireGuard
- حالت تونل‌در‌تونل (`gool`)
- استتار ترافیک
- اتصال مجدد خودکار
- پراکسی محلی SOCKS5
- پشتیبانی از Linux، Windows، macOS و Android (Termux)

## ساخت (Build)

### پیش‌نیازها

- Rust (آخرین نسخه پایدار)
- کامپایلر C/C++
- CMake

پوشه‌ی `quiche` باید در کنار پوشه‌ی `aether` قرار داشته باشد:

```text
<repo>/
  aether/
  quiche/
```

ساخت پروژه:

```bash
cargo build --release
```

فایل نهایی:

```text
target/release/aether
```

## اجرا

برنامه را اجرا کنید:

```bash
./target/release/aether
```

در زمان اجرا، Aether از شما موارد زیر را می‌پرسد:

* پروتکل
* پروفایل استتار
* پورت محلی
* حالت اسکن

پس از اجرا، پراکسی SOCKS5 روی آدرس زیر در دسترس خواهد بود:

```text
127.0.0.1:10808
```

نمونه تست:

```bash
curl -x socks5h://127.0.0.1:10808 https://www.cloudflare.com/cdn-cgi/trace
```

## پروتکل‌های پشتیبانی‌شده

### MASQUE (پیشنهادی)

ترافیک را از طریق HTTP/3 (QUIC) یا HTTP/2 (TLS) عبور می‌دهد تا تا حد ممکن شبیه ترافیک عادی HTTPS باشد.

### WireGuard

یک تونل سریع، سبک و کم‌هزینه که برای شبکه‌هایی با بازرسی کمتر مناسب است.

### تونل‌در‌تونل (`gool`)

یک اتصال WireGuard که داخل یک اتصال WireGuard دیگر اجرا می‌شود و یک لایه رمزنگاری اضافه فراهم می‌کند.
